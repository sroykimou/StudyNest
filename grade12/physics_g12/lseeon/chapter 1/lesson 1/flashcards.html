រូបវិទ្យា ថ្នាក់ទី ១២ • ត្រៀមប្រឡងបាក់ឌុប

មេរៀនទី ១៖ ទែម៉ូឌីណាមិច

រូបមន្តរកចំនួនម៉ូលនៃឧស្ម័ន

មានរូបមន្ត : \( n = \dfrac{m}{M} = \dfrac{N}{N_{\text{A}}} = \dfrac{V}{V_{\text{m}}} \)

  • \(n\):ចំនួនម៉ូល  ( mol )
  • \(m\):ម៉ាសនៃឧស្ម័ន  ( kg )
  • \(M\):ម៉ាសម៉ូល  ( kg/mol )
  • \(N\):ចំនួនម៉ូលេគុលសរុប  ( ម៉ូលេគុល )
  • \(N_{\text{A}}\):ចំនួនអាវ៉ូហ្គាត្រូ  \( ( N_A = 6.21 \times 10^{23} \) ម៉ូលេគុល/mol \()\)
  • \(V\):មាឌឧស្ម័ន  ( m³ )
  • \(V_{\text{m}}\):មាឌម៉ូល  ( m³/mol )
  • •បើនៅក្នុងលក្ខណ្ឌធម្មតា (STP) : \( V_{\text{m}} = 22.4 \text{ L/mol} = 22.4 \times 10^{-3} \text{ m}^3\text{/mol} \)
  • •បើនៅក្នុងលក្ខណ្ឌបន្ទប់ : \( V_{\text{m}} = 24 \text{ L/mol} = 24 \times 10^{-3} \text{ m}^3\text{/mol} \)

រូបមន្តបន្ថែមសំខាន់ៗ

1រកម៉ាស់នៃឧស្ម័ន
រូបមន្ត : \( m = m_0 N \) នាំឲ្យ : \( m_0 = \dfrac{m}{N} \)
  • \(m_0\) : ជាម៉ាសមួយម៉ូលេគុលឧស្ម័នឬម៉ាសម៉ូលេគុលនីមួយៗនៃឧស្ម័ន គិតជា ( kg )
2ទំនាក់ទំនងរវាងម៉ាសម៉ូលេគុលស្ម័នឬម៉ាសម៉ូលេគុលនីមួយៗនៃឧស្ម័ននឹងម៉ាសម៉ូល
រូបមន្ត : \( M = m_0 \times N_{\text{A}} \) នាំឲ្យ : \( m_0 = \dfrac{M}{N_{\text{A}}} \)
ឈ្មោះនិងតម្លៃម៉ាសម៉ូលនៃឧស្ម័នមួយចំនួន
ឧស្ម័នអ៊ីដ្រូសែន : MH₂ = 2 g/mol = 2 × 10−3 kg/mol
ឧស្ម័នអេល្យូម : MHe = 4 g/mol = 4 × 10−3 kg/mol
ឧស្ម័ននីត្រូសែន : MN₂ = 28 g/mol = 28 × 10−3 kg/mol
ឧស្ម័នអុកស៊ីសែន : MO₂ = 32 g/mol = 32 × 10−3 kg/mol
ឧស្ម័នកាបូនឌីអុកស៊ីត : MCO₂ = 44 g/mol = 44 × 10−3 kg/mol
3រូបមន្តបម្លែងសីតុណ្ហភាព
រូបមន្ត : T = t + 273
  • T : ជាសីតុណ្ហភាពដាច់ខាតបម្រែ​ក្នុង​ (K)
  • t : ជាសីតុណ្ហភាពសែលស្យូស គិតជា (°C)

រូបមន្តមាឌសំខាន់ៗ

1រូបមន្តមាឌ  \(( V )\) :
  • •សុីឡាំង  : \( V = Ah = \pi r^2 h = \dfrac{\pi d^2}{4}h \)
  • •គូប  :\( V = A \cdot a = a^3 \)
  • •ប្រលេពីប៉ែតកែង  :\( V = a \times b \times c \)
  • •ស្វ៊ែរ  :\( V = \dfrac{4}{3}\pi R^3 = \dfrac{\pi d^3}{6} \)
2ម៉ាសមាឌ  \( ( \rho ) \) :
  • •រូបមន្ត  :\( \rho = \dfrac{m}{V} \)
  • \(\rho\):ជាម៉ាសមាឌ គិតជា ( kg/m³ )
  • \(m\):ម៉ាសនៃឧស្ម័ន  ( kg )
  • \(V\):មាឌនៃឧស្ម័ន  ( m³ )
3ដង់សុីតេមាឌ  \( ( D ) \) :
  • •រូបមន្ត  :\( D = \dfrac{N}{V} \)
  • \(D\):ជាដង់សុីតេមាឌ គិតជា ( ម៉ូលេគុល/m³ )
  • \(N\):ចំនួនម៉ូលេគុលសរុប  ( ម៉ូលេគុល )
  • \(V\):មាឌនៃឧស្ម័ន  ( m³ )

១. សមីការឧស្ម័នបរិសុទ្ធ

រូបមន្ត : \( PV = nRT = Nk_{\text{B}}T = \dfrac{2}{3}NK_{\text{av}} \)
  • \(P\):ជាសម្ពាធឧស្ម័ន គិតជា ( Pa ) ឬ ( N/m² )
  • \(n\):ចំនួនម៉ូល ( mol )
  • \(V\):ជាមាឌឧស្ម័ន គិតជា ( m³ )
  • \(T\):ជាសីតុណ្ហភាព គិតជា ( K )
  • \(N\):ជាចំនួនម៉ូលេគុលឧស្ម័ន គិតជា ( ម៉ូលេគុល )
  • \(R\):ជាថេរឧស្ម័នឬថេរសកលនៃឧស្ម័ន  \( ( R = 8.31 \text{ J/mol.K} ) \)
  • \(k_{\text{B}}\):ជាថេរប៊ុលស្ម័ន  \( ( k_{\text{B}} = 1.38 \times 10^{-23} \text{ J/K} ) \)
  • \(K_{\text{av}}\):ជាថាមពលសុីនេទិចមធ្យម គិតជា ( J )
🔴 ចំណាំ : \( R = k_{\text{B}} \times N_{\text{A}} = 1.38 \times 10^{-23} \times 6.02 \times 10^{23} = 8.31 \text{ J/mol.K} \)

២. ថាមពលសុីនេទិច

  • • រូបមន្តថាមពលសុីនេទិចមធ្យម : \( K_{\text{av}} = \dfrac{3}{2}k_{\text{B}}T = \dfrac{1}{2}m_0 v^2 \)
  • • រូបមន្តថាមពលសុីនេទិចមធ្យម : \( PV = \dfrac{2}{3}NK_{\text{av}} \Rightarrow K_{\text{av}} = \dfrac{3PV}{2N} \)
  • • រូបមន្តថាមពលស្វ័យសរុប : \( K = NK_{\text{av}} = \dfrac{3}{2}Nk_{\text{B}}T = \dfrac{3}{2}nRT = \dfrac{3}{2}PV \)
  • \(K\):ជាថាមពលសុីនេទិចសរុប គិតជា ( J )
  • \(K_{\text{av}}\):ជាថាមពលសុីនេទិចមធ្យម គិតជា ( J )

៣. ល្បឿនប្រសិទ្ធ ឬ ឫសការេនៃការេល្បឿនមធ្យមរបស់ម៉ូលេគុល ( \(v_{\text{rms}}\) )

  • • រូបមន្ត : \( v_{\text{rms}} = \sqrt{\dfrac{3k_{\text{B}}T}{m_0}} = \sqrt{\dfrac{3RT}{M}} = \sqrt{\dfrac{3P}{\rho}} = \sqrt{\langle v^2 \rangle_{\text{av}}} \)
  • ដែល : \(v_{\text{rms}}\):ល្បឿនប្រសិទ្ធ គិតជា ( m/s )

៤. សមីការប្រែប្រួលភាពនៃឧស្ម័នបរិសុទ្ធ

បើឧស្ម័នប្រែប្រួលភាពពី 1 ទៅ 2 គេបានៈ

  • •នៅភាព 1 ភាពដើម  :\( P_1 V_1 = nRT_1 \quad (1) \)
  • •នៅភាព 2 ភាពស្រស់  :\( P_2 V_2 = nRT_2 \quad (2) \)

យក ( 1 ) ធៀប ( 2 ) គេបានៈ : \( \dfrac{P_1 V_1}{P_2 V_2} = \dfrac{nRT_1}{nRT_2} \)

អាចសរសេរបានៈ \( \dfrac{P_1 V_1}{P_2 V_2} = \dfrac{T_1}{T_2} \) ឬ \( \dfrac{P_1 V_1}{T_1} = \dfrac{P_2 V_2}{T_2} \) ហៅថា សមីការប្រែប្រួលភាពនៃឧស្ម័នបរិសុទ្ធ ។
1ច្បាប់ប៊យ - ម៉ារ្យុត  ( លំនាំអុីសូទែម  \( T_1 = T_2 = \text{Constant} \) )
  • •នៅភាព 1 ភាពដើម  :\( P_1 V_1 = nRT \quad (1) \)
  • •នៅភាព 2 ភាពស្រេច  :\( P_2 V_2 = nRT \quad (2) \)
យក ( 1 ) ធៀប ( 2 ) គេបានៈ \( \dfrac{P_1 V_1}{P_2 V_2} = \dfrac{nRT}{nRT} \Rightarrow \) \( P_1 V_1 = P_2 V_2 \)
2ច្បាប់សាល  ( លំនាំអុីសូបារ  \( P_1 = P_2 = \text{Constant} \) )
  • •នៅភាព 1 ភាពដើម  :\( P V_1 = nRT_1 \quad (1) \)
  • •នៅភាព 2 ភាពស្រស់  :\( P V_2 = nRT_2 \quad (2) \)
យក ( 1 ) ធៀប ( 2 ) គេបាន : \( \dfrac{PV_1}{PV_2} = \dfrac{nRT_1}{nRT_2} \Rightarrow \) \( \dfrac{V_1}{V_2} = \dfrac{T_1}{T_2} \)
3ច្បាប់កេលុយសាក់  ( លំនាំអុីសូករ  \( V_1 = V_2 = \text{Constant} \) )
  • •នៅភាព 1 ភាពដើម  :\( P_1 V = nRT_1 \quad (1) \)
  • •នៅភាព 2 ភាពស្រេច  :\( P_2 V = nRT_2 \quad (2) \)
យក ( 1 ) ធៀប ( 2 ) គេបាន : \( \dfrac{P_1 V}{P_2 V} = \dfrac{nRT_1}{nRT_2} \Rightarrow \) \( \dfrac{P_1}{P_2} = \dfrac{T_1}{T_2} \)
  • \(P_1\):ជាសម្ពាធដើម គិតជា ( Pa )
  • \(P_2\):ជាសម្ពាធស្រេច គិតជា ( Pa )
  • \(V_1\):ជាមាឌដើម គិតជា ( m³ )
  • \(V_2\):ជាមាឌស្រេច គិតជា ( m³ )
  • \(T_1\):ជាសីតុណ្ហភាពដើម គិតជា ( K )
  • \(T_2\):ជាសីតុណ្ហភាពស្រេច គិតជា ( K )

៥. សម្ពាធលើផ្ទៃខាង

សម្ពាធ គឺជាកម្លាំងសង្កត់ក្នុងមួយខ្នាតផ្ទៃ ។

គេសន្មតថា ធុងផ្ទុកឧស្ម័នមួយមានរាងជាគូបដែលមានទ្រនុង \(L\) ម៉ូលេគុលនឺមួយៗ មានម៉ាស \(m_0\) ។

រូបមន្ត : \( P = \dfrac{F}{A} \)

ក. កម្លាំងម៉ូលេគុលនីមួយៗមានលើផ្ទៃខាង

រូបមន្ត : \( F_0 = \dfrac{\Delta p}{A} \)
  • •បរិមាណចលនា និងអាំពុលសាស្យុង :\( F_0 \cdot t = \Delta p \)\(\Rightarrow\)\( F_0 = \dfrac{\Delta p}{\Delta t} \)
  • •ករណីទង្គិចស្ទក់ :\( \Delta p = m_0 v \)\(\Rightarrow\)\( F_0 = \dfrac{m_0 v}{\Delta t} \)
  • •រយៈពេល :\( t = \dfrac{L}{v} \)
  • •ករណីទង្គិចខ្ទាត :\( \Delta p = 2m_0 v \)\(\Rightarrow\)\( F_0 = \dfrac{2m_0 v}{\Delta t} \)
  • •រយៈពេល :\( t = \dfrac{2L}{v} \)

ខ. កម្លាំងសរុបដែលមូលេគុលមានលើផ្ទៃខាង

  • •រូបមន្ត :\( F = NF_0 \)
  • •ករណីទង្គិចស្ទក់ :\( F = \dfrac{Nm_0 v}{\Delta t} \)
  • •ករណីទង្គិចខ្ទាត :\( F = \dfrac{2Nm_0 v}{\Delta t} \)
  • \(\Delta p\):ជាប្រែប្រួលបរិមាណចលនា គិតជា ( kg·m/s )
  • \(\Delta t\):ជាប្រែប្រួលរយៈពេល គិតជា ( s )
  • \(N\):ជាចំនួនម៉ូលេគុលនៃឧស្ម័ន គិតជា ( ម៉ូលេគុល )
  • \(F\):ជាកម្លាំងសរុបរបស់ម៉ូលេគុលឧស្ម័នមានលើផ្ទៃ គិតជា ( N )
  • \(F_0\):ជាកម្លាំងដែលម៉ូលេគុលនីមួយៗមានលើផ្ទៃ គិតជា ( N )
